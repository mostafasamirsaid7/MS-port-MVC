## Project Overview

**Mostafa Said Portfolio** is a full-featured portfolio website showcasing projects, blog posts, testimonials, and events. Built with ASP.NET Core MVC with:

- **Database**: PostgreSQL with Entity Framework Core + Dapper
- **.NET Version**: .NET 9
- **Authentication**: ASP.NET Identity
- **Localization**: English & Arabic support
- **ORM**: EF Core for Identity, Dapper for custom queries
